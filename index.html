Cara Kerja Internet

Internet bekerja dengan menghubungkan komputer dan jaringan komputer di seluruh dunia melalui protokol TCP/IP. Protokol ini membagi data menjadi paket-paket kecil, kemudian mengumpulkannya menjadi pesan yang dikirimkan dari satu komputer ke komputer lain.

Cara Kerja HTTP dan HTTPS

HTTP dan HTTPS adalah protokol yang digunakan untuk mentransfer data di internet. Perbedaan utama antara keduanya adalah HTTPS menggunakan enkripsi untuk mengamankan data, sedangkan HTTP tidak.

Cara Kerja Domain

Nama domain adalah nama unik yang digunakan untuk mengidentifikasi situs web di internet. Nama domain bekerja dengan cara meringkas alamat IP yang terdiri dari gabungan angka yang unik, kemudian disimpan menjadi satu kesatuan di dalam DNS Server.

Cara Kerja Hosting

Hosting adalah layanan yang menyediakan tempat untuk menyimpan dan mengakses data website secara online. Data-data tersebut dapat berupa video, gambar, email, script, aplikasi, dan database.

Cara Kerja DNS

DNS (Domain Name System) bekerja dengan mengubah nama domain menjadi alamat IP.

Cara Kerja Browser

Cara kerja web browser adalah dengan mengambil informasi dari suatu situs dan mentransfernya melalui Hypertext Transfer Protocol (HTTP) yang menjadi media transmisi teks, gambar, dan video antar situs.
